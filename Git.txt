1. git init: para inicializar el git en la carpeta seleccionada
2. git status: para ver el estado de los archivos
3. git log: para ver el historial de commits
4. git add: para añadir un archivo 
5. git commit: para comentar el archivo y poder subirlo
6. git remote add origin https://github.com/Luis9459/HTML-CSS-2022.git: para que git sepa donde subir los archivos
7. git puch -u origin main: para enlazarlo con github
8. git push: cuando ya esté enlazado y subirlo
